## 🛠️ Prerequisites
Before you jump in, make sure you have the following ready:
1. **Node.js** (v16 or higher recommended)
2. **Discord Bot Token** (Get this from the Discord Developer Portal)
3. Your **Discord User ID** (For bot owner commands)
4. That all

## 🚀 How to Use

Getting started is easy as hell, run these cmd on ur cmd:
```
git clone https://github.com/meh2025/Santaa-discordBot.git
cd Santaa-discordBot
npm install
```

We have two way to run this noel ahh bot:
1. running via cmd
```bash
npm run dev # for development

npm run start # for daily use
```
2. running via shell: idk just run the santaa.sh bro 😭 😭
